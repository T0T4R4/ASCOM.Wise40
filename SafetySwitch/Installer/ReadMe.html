Abstract

This is an ASCOM SafetyMonitor driver for the Wise Observatory. It monitors the observatory's main SafetySwitch.

The driver's IsSafe method should be called before any of the observatory's computer-controlled parts is controlled.

Hardware

The Wise 40" Observatory has a main SafetySwitch at the back of the observatory's original console. It enables/disables computer control of the various components (telescope, dome, focus, etc.). The main use of the switch is to prevent computer control while maintenance tasks are being performed at the observatory.

Software

If the driver's IsSafe method returns false the controlling method should throw an InvalidOperationException, as prescribed by the ASCOM standard for operations that cannot be performed at the moment.

Setup

The driver has no configurable parameters.

ASCOM Compliance

The driver passes the ASCOM 6.2 Compliance test.
