Abstract

This is an ASCOM SafetyMonitor driver for the Wise Observatory. It defines a set of environment parameters and values that are monitored in order to decide whether it is safe to open the observatory.

The driver's IsSafe method can be called:

  • Before starting an observing session, to decide whether to start it or not
  • While the session is in progress, to decide whether to continue it or not.

Hardware

The weather data is gathered from both one of the Boltwood CloudSensorII and from the Davis VantagePro2 weather stations available at the Wise Observatory.

Software

This is a wrapper ASCOM driver. It uses the weather stations' own ASCOM drivers to retrieve the actual data. It then checks if any of the monitored parameters excedes the maximal values defined in the driver's profile.

Setup

The driver has an ASCOM compliant Setup window where the monitored parameters and their maximal values are defined.

ASCOM Compliance

The driver passes the ASCOM 6.2 Compliance test.
