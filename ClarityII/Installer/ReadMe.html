Abstract.

This is an ASCOM driver for the CloudSensorII from Boltwood Systems.

It presents the sensor-data gathered from a devicein an ASCOM compliant interface. The device can be connected either to the local computer or to a remote one.

Method

The driver is based on the ClarityII single-line-data-file (new format) feature. This is described in Section 17.1 of the Cloud Sensor II User’s Manual.

The ClarityII software can be configured (via its Setup) to save the raw data gathered from the sensors in an external file with a very specific format. The data-file can be saved on a local folder or in one that gets mapped to/from other computers. It is updated every two seconds.

The ASCOM driver reads the data-file and presents its contents via an ASCOM ObservingConditions object.

Setup

The driver has an ASCOM compliant Setup form. The path to the data-file is entered and saved in the driver's ASCOM profile.
