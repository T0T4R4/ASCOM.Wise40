Abstract

This is an ASCOM Focuser driver for the Wise Observatory.

The driver's IsSafe method should be called before any of the observatory's computer-controlled parts is controlled.

Hardware

The Wise 40" Focuser has:

  • A Up/Down motor
  • A multi-turn absolute encoder

Software

The driver provides the ASCOM:

  • Position property
  • The Move(position) method

Setup

The driver has no configurable parameters.

ASCOM Compliance

The driver passes the ASCOM 6.2 Compliance test.
