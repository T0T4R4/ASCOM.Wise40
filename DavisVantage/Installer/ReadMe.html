Abstract

This is an ASCOM ObservingConditions driver for the VantagePro weather station of the Wise40 instrument at the Wise Observatory. We needed to write this driver since the actual station is hooked-up to a computer different from the one driving the 40" telescope.

Hardware

The VantagePro weather station has a serial port interface and is hooked-up to a serial port on the weather-pc computer in the main dome. The computer driving the dome (dome-ctlr) has no access to this port.

Software

The WeatherLink software, provided by the Davis Company and running on weather-pc, was configured to periodically (every 10 minutes) produce an HTML report that dumps the values for all its internal variables (using the Weather_Wise40_Vantage_Pro.htx template file).

The report is produced in a local folder on weather-pc.

This driver parses the HTML report file (mapped to a folder on dome-ctlr) and presents it as an ASCOM ObservationConditions object.

Setup

The driver has an ASCOM compliant Setup window where the path to the HTML report file can be selected.

ASCOM Compliance

The driver passes the ASCOM 6.2 Compliance test.
